Vollstaendige Anleitung

Diese Anleitung erklaert CV-Studio Schritt fuer Schritt - von ganz vorne. Sie ist fuer absolute Einsteiger geschrieben: jeder Knopf, jede Funktion und jeder Handgriff wird erklaert. Damit Sie sich nie verloren fuehlen.

1. Was ist CV-Studio?

CV-Studio ist eine kostenlose Anwendung, mit der Sie professionelle Lebenslaeufe direkt im Browser erstellen, bearbeiten und als PDF exportieren koennen. Sie brauchen kein Konto, keine Installation und keine Internetverbindung (ausser fuer den PDF-Export und das Laden der Schriftarten).

  • Mehrere Lebenslaeufe: Legen Sie beliebig viele eigenstaendige Lebenslaeufe an (zum Beispiel je einen pro Bewerbungsziel).
  • Live-Vorschau: Jede Aenderung erscheint sofort rechts in der A4-Vorschau.
  • 10 Vorlagen und 11 Schriftarten: Gestalten Sie das Aussehen mit wenigen Klicks.
  • PDF-Export: Erzeugen Sie ein druckfertiges PDF mit anklickbaren Links.
  • Ihre Daten bleiben bei Ihnen: Alles wird lokal in Ihrem Browser gespeichert (siehe Abschnitt 15).
Gut zu wissenSie koennen nichts kaputt machen. Probieren Sie ruhig alles aus - mit JSON-Export (Abschnitt 14) sichern Sie Ihren Stand jederzeit.

2. Erste Schritte

  1. Oeffnen Sie die Anwendung: entweder die Datei index.html per Doppelklick oder die im Web veroeffentlichte Adresse.
  2. Beim allerersten Start sehen Sie einen Beispiel-Lebenslauf (Demo). So erkennen Sie sofort, wie ein fertiger Lebenslauf aussieht.
  3. Legen Sie Ihren eigenen Lebenslauf an: oben links auf + Neu klicken (siehe Abschnitt 4). Oder bearbeiten Sie die Demo einfach mit Ihren Daten.
  4. Fuellen Sie die Bereiche links aus. Die Vorschau rechts aktualisiert sich automatisch.
  5. Waehlen Sie eine Vorlage und eine Schrift, die Ihnen gefallen (Abschnitte 7 und 8).
  6. Exportieren Sie das Ergebnis als PDF (Abschnitt 13).

3. Die Oberflaeche im Ueberblick

Die Anwendung besteht aus drei Bereichen:

  • Obere Leiste: Hier verwalten Sie Lebenslaeufe (Auswahl, Neu, Duplizieren, Umbenennen, Loeschen), waehlen Vorlage, Schrift und Textfarbe und finden die Knoepfe fuer JSON-Import, JSON-Export und PDF-Export. Ganz rechts steht ein Speicher-Hinweis (zum Beispiel "Gespeichert").
  • Editor (links): Hier geben Sie Ihre Inhalte ein. Oben befinden sich Reiter (Tabs): Persoenlich, Skills, Erfahrung, Bildung, Projekte, Sprachen, Sonstiges.
  • Vorschau (rechts): Zeigt Ihren Lebenslauf in Echtzeit als A4-Seite. Gestrichelte Linien zeigen, wo eine Seite endet und die naechste beginnt.
Automatisches SpeichernSie muessen nichts manuell speichern. Sobald oben rechts "Gespeichert" steht, ist Ihr aktueller Stand im Browser gesichert.

4. Mehrere Lebenslaeufe verwalten

In der oberen Leiste finden Sie ganz links den Bereich Lebenslauf mit einem Auswahlfeld und vier Knoepfen.

Einen neuen Lebenslauf anlegen

  1. Klicken Sie auf + Neu.
  2. Geben Sie einen Namen ein (zum Beispiel "Bewerbung Data Engineer") und bestaetigen Sie.
  3. Es wird ein leerer Lebenslauf angelegt und sofort aktiv. Fuellen Sie ihn nun aus.

Zwischen Lebenslaeufen wechseln

Oeffnen Sie das Auswahlfeld und waehlen Sie den gewuenschten Lebenslauf. Die Anwendung laedt sofort dessen Inhalte.

Duplizieren, Umbenennen, Loeschen

  • Duplizieren: erstellt eine unabhaengige Kopie des aktuellen Lebenslaufs (zum Beispiel um eine Variante auszuprobieren).
  • Umbenennen: aendert den Namen des aktuellen Lebenslaufs.
  • Loeschen: entfernt den aktuellen Lebenslauf endgueltig. Es erscheint eine Sicherheitsabfrage. Der Knopf ist gesperrt, wenn nur noch ein Lebenslauf vorhanden ist.
Achtung beim LoeschenLoeschen kann nicht rueckgaengig gemacht werden. Wenn Sie unsicher sind, exportieren Sie den Lebenslauf vorher als JSON (Abschnitt 14).

5. Inhalte bearbeiten (alle Bereiche)

Im Editor links waehlen Sie oben einen Reiter. Jede Aenderung erscheint sofort in der Vorschau.

Persoenlich

Hier stehen Ihre Stammdaten: Name, Berufsbezeichnung, Adresse, E-Mail, Telefon, LinkedIn, GitHub, Portfolio/Website, Geburtsdatum, Geburtsort, Staatsbuergerschaft und Familienstand. Tragen Sie ein, was Sie zeigen moechten. Leere Felder werden in der Vorschau einfach weggelassen.

Skills (Kenntnisse)

  1. Klicken Sie auf + Kategorie hinzufuegen (zum Beispiel "Programmierung").
  2. Klicken Sie bei der Kategorie auf Bearbeiten.
  3. Aendern Sie bei Bedarf den Kategorienamen.
  4. Geben Sie unten im Feld einen Skill ein (zum Beispiel "Python") und druecken Sie Enter oder das +. Wiederholen Sie das fuer weitere Skills.
  5. Ein Skill laesst sich ueber das kleine x neben dem Eintrag wieder entfernen.
  6. Optional: stellen Sie je Skill eine Sternebewertung ein (Auswahl rechts neben dem Skill, 1 bis 5 Sterne oder "ohne"). Die Sterne werden in der Vorlage "Profil Klassisch" angezeigt; andere Vorlagen ignorieren sie.

Erfahrung (Berufserfahrung)

  1. Klicken Sie auf + Eintrag hinzufuegen.
  2. Klicken Sie auf Bearbeiten und fuellen Sie Position, Unternehmen, Ort, Von und Bis aus.
  3. Fuegen Sie unter "Aufgaben" mit + Punkt einzelne Stichpunkte hinzu. Halten Sie diese kurz (siehe Abschnitt 16).

Bildung (Ausbildung)

Wie bei der Erfahrung: Abschluss, Institution, Ort, Von, Bis und optionale Anmerkungen.

Projekte

  1. Klicken Sie auf + Eintrag hinzufuegen und dann Bearbeiten.
  2. Tragen Sie Projekttitel und Art/Kontext ein.
  3. Optional: tragen Sie unter Projekt-Link die Adresse des Projekts ein (zum Beispiel Ihr GitHub-Repository). Dann erscheint im Lebenslauf ein dezenter Knopf "Ansehen", der im PDF anklickbar ist.
  4. Fuegen Sie unter "Tech-Stack" die verwendeten Technologien hinzu (Eingabe und Enter).
  5. Fuegen Sie unter "Ergebnisse" kurze Stichpunkte hinzu.

Sprachen

Fuegen Sie Sprache und Niveau hinzu (zum Beispiel "Deutsch" / "C1").

Sonstiges (Interessen)

Kurze Eintraege wie Interessen, Fuehrerschein oder Ehrenamt.

Eintraege schliessenMit erneutem Klick auf Bearbeiten klappen Sie eine Karte wieder zu. Das schafft Uebersicht.

6. Foto hochladen

  1. Gehen Sie in den Reiter Persoenlich.
  2. Klicken Sie auf Foto waehlen und waehlen Sie eine Bilddatei von Ihrem Geraet.
  3. Unter Foto-Darstellung waehlen Sie zwischen Rund und Eckig.
  4. Mit dem Regler Foto-Zoom vergroessern Sie den Bildausschnitt (100 bis 250 Prozent), mit Foto-Position vertikal verschieben Sie den Ausschnitt nach oben oder unten - so sitzt das Gesicht perfekt im Rahmen. Die Originaldatei bleibt dabei unveraendert.
  5. Mit Entfernen loeschen Sie das Foto wieder.
Tipp zum BildVerwenden Sie ein moeglichst quadratisches, gut belichtetes Portraitfoto. Das Bild wird direkt im Browser gespeichert und nirgendwohin hochgeladen.

7. Vorlagen (Designs)

Oben in der Leiste waehlen Sie unter Vorlage aus 10 Designs. Der Inhalt bleibt gleich, nur das Aussehen aendert sich.

8. Schriftart waehlen

Unter Schrift waehlen Sie aus 11 Schriftarten (zum Beispiel Inter, Arial, Calibri, Georgia, Times New Roman). Die Vorschau aktualisiert sich sofort. Serifenschriften (Georgia, Times, EB Garamond) wirken klassisch, serifenlose (Inter, Arial, Lato) modern.

9. Textfarbe aendern

Standardmaessig ist der Text schwarz - das ist am besten lesbar und wird von Personalverantwortlichen erwartet. Sie koennen die Farbe aber anpassen:

  1. Klicken Sie oben in der Leiste bei Textfarbe auf das Farbfeld.
  2. Waehlen Sie eine Farbe. Die Vorschau aktualisiert sich sofort.
  3. Mit dem Knopf Schwarz setzen Sie die Farbe jederzeit auf den Standard zurueck.
Lesbarkeit beachtenSehr helle Farben (zum Beispiel Hellgrau oder Gelb) sind auf weissem Papier schwer lesbar und wirken unprofessionell. Bleiben Sie bei Schwarz oder einem dunklen Farbton. Die Akzentfarben der Ueberschriften aendern sich hierdurch nicht.

10. Portfolio, LinkedIn, GitHub und Projekt-Links

  • Portfolio/Website: Wenn Sie im Reiter Persoenlich eine Website eintragen, erscheint sie als gut sichtbare Schaltflaeche "Portfolio" - sie faellt auf, statt als lange Adresse unterzugehen.
  • LinkedIn und GitHub: werden als kompakte Symbole angezeigt (nicht als lange Adresse). Das spart Platz und sieht aufgeraeumt aus.
  • Projekt-Links: siehe Abschnitt 5 (Projekte). Erscheinen als kleiner Knopf "Ansehen".

Alle diese Links bleiben im exportierten PDF anklickbar.

11. Reihenfolge per Drag and Drop

  1. Fahren Sie im Editor mit der Maus ueber eine Karte (zum Beispiel eine Berufserfahrung).
  2. Klicken und halten Sie die Karte (am Griff-Symbol links) und ziehen Sie sie nach oben oder unten.
  3. Lassen Sie an der gewuenschten Position los. Die Reihenfolge im Lebenslauf passt sich sofort an.
EmpfehlungStellen Sie die wichtigsten und aktuellsten Eintraege nach oben - sie werden zuerst gelesen.

12. A4-Seiten und Seitenumbrueche

Die Vorschau zeigt Ihren Lebenslauf als echte A4-Seiten. An jeder Seitengrenze sehen Sie eine dezente gestrichelte Linie mit der Beschriftung "Seite 2", "Seite 3" und so weiter.

  • Damit kein Eintrag mitten im Text abgeschnitten wird, schiebt die Anwendung Zeilen, die genau auf einer Seitengrenze liegen wuerden, automatisch auf die naechste Seite - in allen Vorlagen.
  • Das exportierte PDF entspricht exakt der Vorschau - inklusive dieser sauberen Umbrueche.
Lebenslauf zu lang?Kuerzen Sie Stichpunkte, entfernen Sie aeltere Eintraege oder reduzieren Sie die Anzahl der Projekte. Ein bis zwei Seiten sind ideal.

13. Als PDF exportieren

  1. Klicken Sie oben rechts auf PDF exportieren.
  2. Warten Sie kurz (der Knopf zeigt "Bitte warten..."). Die Anwendung erzeugt ein mehrseitiges A4-PDF.
  3. Das PDF wird automatisch heruntergeladen. Der Dateiname enthaelt den Namen Ihres Lebenslaufs.
  • Die Links (E-Mail, LinkedIn, GitHub, Portfolio, Projekte) bleiben im PDF anklickbar.
  • Das PDF entspricht der Vorschau. Pruefen Sie es vor dem Versenden trotzdem einmal.
InternetverbindungFuer den PDF-Export und das korrekte Laden der Schriftarten ist eine Internetverbindung noetig.

14. Sichern: JSON-Export und -Import

Mit JSON-Dateien sichern Sie Ihre Lebenslaeufe als Datei - die zuverlaessigste Methode gegen Datenverlust und zum Wechsel auf ein anderes Geraet.

Exportieren (sichern)

  1. Waehlen Sie oben den gewuenschten Lebenslauf aus.
  2. Klicken Sie auf JSON exportieren. Es wird eine .json-Datei heruntergeladen.
  3. Bewahren Sie diese Datei sicher auf (zum Beispiel in einem Cloud-Ordner).

Importieren (wiederherstellen)

  1. Klicken Sie auf JSON importieren.
  2. Waehlen Sie Ihre .json-Datei aus.
  3. Es wird daraus ein neuer Lebenslauf angelegt (vorhandene Lebenslaeufe bleiben unveraendert).
EmpfehlungExportieren Sie Ihren fertigen Lebenslauf einmal als JSON und bewahren Sie die Datei auf. So koennen Sie ihn jederzeit auf jedem Geraet wiederherstellen.

15. Wo werden meine Daten gespeichert?

Das ist wichtig zu verstehen, damit Sie keine Daten verlieren:

  • Ihre Lebenslaeufe werden im lokalen Speicher Ihres Browsers gespeichert (LocalStorage), nicht auf einem Server.
  • Der Speicher ist an einen Browser und eine Adresse gebunden. Auf einem anderen Computer, in einem anderen Browser oder unter einer anderen Webadresse sehen Sie Ihre Daten daher nicht automatisch.
  • Wenn Sie die Browserdaten loeschen (zum Beispiel den Verlauf inklusive Webseitendaten), werden auch Ihre Lebenslaeufe geloescht.
So vermeiden Sie DatenverlustExportieren Sie wichtige Lebenslaeufe als JSON-Datei (Abschnitt 14) und bewahren Sie sie auf. Das ist die einzige sichere Methode fuer eine dauerhafte Sicherung und fuer den Wechsel zwischen Geraeten.

Beim ersten Start in einem leeren Browser zeigt die App einen Beispiel-Lebenslauf, damit sie nie leer wirkt. Ihre eigenen Daten werden dadurch niemals ueberschrieben.

16. Tipps fuer einen ueberzeugenden Lebenslauf

  • Kurz und praegnant: ein Gedanke pro Stichpunkt, moeglichst eine Zeile. Vermeiden Sie lange Textbloecke.
  • Ergebnisorientiert: nennen Sie Resultate und Zahlen (zum Beispiel "Ladezeit um 60 Prozent reduziert").
  • Lesbarkeit zuerst: schwarze Schrift auf weissem Grund, klare Schriftart, genug Abstand.
  • Relevanz: stellen Sie die wichtigsten Eintraege nach oben (Drag and Drop, Abschnitt 11).
  • Laenge: ein bis zwei Seiten reichen meist. Weniger ist oft mehr.
  • Pruefen: kontrollieren Sie Rechtschreibung und das fertige PDF vor dem Versenden.

17. Haeufige Fragen und Problemloesung

Auf der veroeffentlichten Webseite ist mein Lebenslauf leer. Warum?
Ihre Daten liegen im lokalen Speicher genau des Browsers und der Adresse, in dem Sie sie erstellt haben (siehe Abschnitt 15). Die Webseite hat einen eigenen, zunaechst leeren Speicher. Loesung: exportieren Sie Ihren Lebenslauf dort als JSON, wo er vorhanden ist, und importieren Sie ihn auf der Webseite (Abschnitt 14).
Meine Daten sind verschwunden. Sind sie verloren?
Meist nicht. Pruefen Sie, ob Sie denselben Browser und dieselbe Adresse verwenden wie beim Erstellen. Pruefen Sie das Auswahlfeld oben - vielleicht ist nur ein anderer Lebenslauf aktiv. Falls Sie eine JSON-Sicherung haben, importieren Sie diese. Wenn Sie die Browserdaten geloescht haben, lassen sich nur Daten aus einer JSON-Sicherung wiederherstellen.
Das PDF schneidet Inhalte ab.
Einzelne Zeilen, die auf einer Seitengrenze liegen wuerden, werden in allen Vorlagen automatisch auf die naechste Seite geschoben. Falls dennoch etwas unguenstig umbricht, kuerzen Sie den betroffenen Eintrag oder verschieben Sie ihn per Drag and Drop.
Die LinkedIn- oder GitHub-Symbole sind im PDF nicht sichtbar.
Die Symbole passen ihre Farbe an die Textfarbe an und werden beim Export korrekt eingefaerbt. Falls ein Symbol auf einem dunklen Hintergrund nicht erscheint, pruefen Sie, ob die Adresse korrekt eingetragen ist, und exportieren Sie erneut.
Kann ich denselben Lebenslauf auf mehreren Geraeten bearbeiten?
Nicht automatisch, da die Daten lokal im Browser liegen. Exportieren Sie den Lebenslauf als JSON und importieren Sie ihn auf dem anderen Geraet (Abschnitt 14).
Muss ich speichern?
Nein. Die App speichert automatisch. Wenn oben rechts "Gespeichert" steht, ist Ihr Stand gesichert. Fuer eine dauerhafte Sicherung nutzen Sie zusaetzlich den JSON-Export.
Die Schrift wirkt grau und schwer lesbar.
Klicken Sie oben bei Textfarbe auf Schwarz, um die gut lesbare Standardfarbe zu setzen (Abschnitt 9).

Complete guide

This guide explains CV-Studio step by step - from the very beginning. It is written for absolute beginners: every button, every feature and every action is explained. So you never feel lost.

1. What is CV-Studio?

CV-Studio is a free application that lets you create, edit and export professional resumes as PDF directly in your browser. You do not need an account, an installation or an internet connection (except for the PDF export and loading the fonts).

  • Multiple resumes: Create as many independent resumes as you like (for example one per application target).
  • Live preview: Every change appears immediately on the right in the A4 preview.
  • 10 templates and 11 fonts: Shape the look with just a few clicks.
  • PDF export: Generate a print-ready PDF with clickable links.
  • Your data stays with you: Everything is stored locally in your browser (see section 15).
Good to knowYou cannot break anything. Feel free to try everything - with the JSON export (section 14) you can save your current state at any time.

2. First steps

  1. Open the application: either the file index.html by double-clicking, or the address published on the web.
  2. On the very first start you see an example resume (demo). This shows you right away what a finished resume looks like.
  3. Create your own resume: click + New at the top left (see section 4). Or simply edit the demo with your own data.
  4. Fill in the sections on the left. The preview on the right updates automatically.
  5. Choose a template and a font you like (sections 7 and 8).
  6. Export the result as PDF (section 13).

3. The interface at a glance

The application consists of three areas:

  • Top bar: Here you manage resumes (selection, New, Duplicate, Rename, Delete), choose the template, font and text color, and find the buttons for JSON import, JSON export and PDF export. On the far right there is a save notice (for example "Saved").
  • Editor (left): Here you enter your content. At the top there are tabs: Personal, Skills, Experience, Education, Projects, Languages, Other.
  • Preview (right): Shows your resume in real time as an A4 page. Dashed lines show where one page ends and the next begins.
Automatic savingYou do not have to save anything manually. As soon as "Saved" appears at the top right, your current state is secured in the browser.

4. Managing multiple resumes

In the top bar, on the far left, you find the Resume area with a selection field and four buttons.

Creating a new resume

  1. Click + New.
  2. Enter a name (for example "Application Data Engineer") and confirm.
  3. An empty resume is created and immediately active. Now fill it in.

Switching between resumes

Open the selection field and choose the desired resume. The application loads its content immediately.

Duplicate, Rename, Delete

  • Duplicate: creates an independent copy of the current resume (for example to try out a variant).
  • Rename: changes the name of the current resume.
  • Delete: removes the current resume permanently. A safety prompt appears. The button is disabled when only one resume is left.
Caution when deletingDeleting cannot be undone. If you are unsure, export the resume as JSON beforehand (section 14).

5. Editing content (all sections)

In the editor on the left you choose a tab at the top. Every change appears immediately in the preview.

Personal

Here is your master data: name, job title, address, e-mail, phone, LinkedIn, GitHub, portfolio/website, date of birth, place of birth, citizenship and marital status. Enter whatever you want to show. Empty fields are simply omitted in the preview.

Skills

  1. Click + Add category (for example "Programming").
  2. Click Edit on the category.
  3. Change the category name if needed.
  4. Enter a skill in the field at the bottom (for example "Python") and press Enter or the +. Repeat this for further skills.
  5. A skill can be removed again via the small x next to the entry.
  6. Optional: set a star rating per skill (selection to the right of the skill, 1 to 5 stars or "none"). The stars are shown in the "Profil Klassisch" template; other templates ignore them.

Experience (work experience)

  1. Click + Add entry.
  2. Click Edit and fill in position, company, location, From and To.
  3. Under "Tasks", add individual bullet points with + Point. Keep them short (see section 16).

Education

Like the experience: degree, institution, location, From, To and optional notes.

Projects

  1. Click + Add entry and then Edit.
  2. Enter the project title and type/context.
  3. Optional: under Project link, enter the address of the project (for example your GitHub repository). Then a subtle "View" button appears in the resume, which is clickable in the PDF.
  4. Under "Tech stack", add the technologies used (type and Enter).
  5. Under "Results", add short bullet points.

Languages

Add language and level (for example "German" / "C1").

Other (interests)

Short entries such as interests, driving license or volunteer work.

Closing entriesBy clicking Edit again you collapse a card. This keeps things tidy.

6. Uploading a photo

  1. Go to the Personal tab.
  2. Click Choose photo and select an image file from your device.
  3. Under Photo display you choose between Round and Square.
  4. With the Photo zoom slider you enlarge the image crop (100 to 250 percent), and with Photo position vertical you move the crop up or down - so the face sits perfectly in the frame. The original file remains unchanged.
  5. With Remove you delete the photo again.
Photo tipUse a portrait photo that is as square as possible and well lit. The image is stored directly in the browser and is not uploaded anywhere.

At the top of the bar you choose from 10 designs under Template. The content stays the same, only the look changes.

8. Choosing a font

Under Font you choose from 11 fonts (for example Inter, Arial, Calibri, Georgia, Times New Roman). The preview updates immediately. Serif fonts (Georgia, Times, EB Garamond) feel classic, sans-serif ones (Inter, Arial, Lato) modern.

9. Changing the text color

By default the text is black - that is the most readable and is expected by recruiters. But you can adjust the color:

  1. At the top of the bar, click the color field at Text color.
  2. Choose a color. The preview updates immediately.
  3. With the Black button you can reset the color to the default at any time.
Mind readabilityVery light colors (for example light gray or yellow) are hard to read on white paper and look unprofessional. Stick to black or a dark shade. The accent colors of the headings do not change through this.

10. Portfolio, LinkedIn, GitHub and project links

  • Portfolio/Website: If you enter a website in the Personal tab, it appears as a clearly visible "Portfolio" button - it stands out instead of getting lost as a long address.
  • LinkedIn and GitHub: are shown as compact icons (not as a long address). This saves space and looks tidy.
  • Project links: see section 5 (Projects). They appear as a small "View" button.

All of these links remain clickable in the exported PDF.

11. Reordering with drag and drop

  1. In the editor, move the mouse over a card (for example a work experience).
  2. Click and hold the card (on the handle icon on the left) and drag it up or down.
  3. Release at the desired position. The order in the resume adjusts immediately.
RecommendationPut the most important and most recent entries at the top - they are read first.

12. A4 pages and page breaks

The preview shows your resume as real A4 pages. At every page boundary you see a subtle dashed line labeled "Page 2", "Page 3" and so on.

  • So that no entry is cut off in the middle of the text, the application automatically pushes lines that would fall exactly on a page boundary to the next page - in all templates.
  • The exported PDF matches the preview exactly - including these clean breaks.
Resume too long?Shorten bullet points, remove older entries or reduce the number of projects. One to two pages are ideal.

13. Exporting as PDF

  1. Click Export PDF at the top right.
  2. Wait a moment (the button shows "Please wait..."). The application generates a multi-page A4 PDF.
  3. The PDF is downloaded automatically. The file name contains the name of your resume.
  • The links (e-mail, LinkedIn, GitHub, portfolio, projects) remain clickable in the PDF.
  • The PDF matches the preview. Still, check it once before sending.
Internet connectionAn internet connection is needed for the PDF export and for loading the fonts correctly.

14. Backups: JSON export and import

With JSON files you save your resumes as a file - the most reliable method against data loss and for switching to another device.

Exporting (saving)

  1. Select the desired resume at the top.
  2. Click Export JSON. A .json file is downloaded.
  3. Keep this file safe (for example in a cloud folder).

Importing (restoring)

  1. Click Import JSON.
  2. Select your .json file.
  3. A new resume is created from it (existing resumes remain unchanged).
RecommendationExport your finished resume once as JSON and keep the file. This way you can restore it at any time on any device.

15. Where is my data stored?

This is important to understand so that you do not lose any data:

  • Your resumes are stored in the local storage of your browser (LocalStorage), not on a server.
  • The storage is bound to one browser and one address. On another computer, in another browser or under a different web address, you therefore do not see your data automatically.
  • If you delete the browser data (for example the history including website data), your resumes are deleted too.
How to avoid data lossExport important resumes as a JSON file (section 14) and keep them. This is the only secure method for a permanent backup and for switching between devices.

On the first start in an empty browser, the app shows an example resume so that it never appears empty. Your own data is never overwritten by this.

16. Tips for a convincing resume

  • Short and concise: one thought per bullet point, ideally one line. Avoid long blocks of text.
  • Results-oriented: state results and numbers (for example "reduced loading time by 60 percent").
  • Readability first: black text on a white background, a clear font, enough spacing.
  • Relevance: put the most important entries at the top (drag and drop, section 11).
  • Length: one to two pages are usually enough. Less is often more.
  • Check: review the spelling and the finished PDF before sending.

17. Frequently asked questions and troubleshooting

On the published website my resume is empty. Why?
Your data is located in the local storage of exactly the browser and the address in which you created it (see section 15). The website has its own, initially empty storage. Solution: export your resume as JSON where it is present, and import it on the website (section 14).
My data has disappeared. Is it lost?
Usually not. Check whether you are using the same browser and the same address as when creating it. Check the selection field at the top - maybe just a different resume is active. If you have a JSON backup, import it. If you deleted the browser data, only data from a JSON backup can be restored.
The PDF cuts off content.
Individual lines that would fall on a page boundary are automatically pushed to the next page in all templates. If something still breaks unfavorably, shorten the affected entry or move it with drag and drop.
The LinkedIn or GitHub icons are not visible in the PDF.
The icons adapt their color to the text color and are colored correctly during export. If an icon does not appear on a dark background, check whether the address is entered correctly, and export again.
Can I edit the same resume on multiple devices?
Not automatically, since the data is stored locally in the browser. Export the resume as JSON and import it on the other device (section 14).
Do I have to save?
No. The app saves automatically. When "Saved" appears at the top right, your state is secured. For a permanent backup, additionally use the JSON export.
The text looks gray and hard to read.
At the top, click Black at the text color to set the easily readable default color (section 9).

Guide complet

Ce guide explique CV-Studio etape par etape - depuis le tout debut. Il est ecrit pour les debutants absolus : chaque bouton, chaque fonction et chaque manipulation est explique. Pour que vous ne vous sentiez jamais perdu.

1. Qu'est-ce que CV-Studio ?

CV-Studio est une application gratuite qui vous permet de creer, modifier et exporter en PDF des CV professionnels directement dans le navigateur. Vous n'avez besoin ni de compte, ni d'installation, ni de connexion internet (sauf pour l'export PDF et le chargement des polices).

  • Plusieurs CV : creez autant de CV independants que vous le souhaitez (par exemple un par objectif de candidature).
  • Apercu en direct : chaque modification apparait immediatement a droite dans l'apercu A4.
  • 10 modeles et 11 polices : composez l'apparence en quelques clics.
  • Export PDF : generez un PDF pret a imprimer avec des liens cliquables.
  • Vos donnees restent chez vous : tout est enregistre localement dans votre navigateur (voir section 15).
Bon a savoirVous ne pouvez rien casser. Essayez tout en toute tranquillite - avec l'export JSON (section 14), vous sauvegardez votre etat a tout moment.

2. Premiers pas

  1. Ouvrez l'application : soit le fichier index.html par double-clic, soit l'adresse publiee sur le web.
  2. Au tout premier demarrage, vous voyez un CV d'exemple (demo). Cela vous montre tout de suite a quoi ressemble un CV termine.
  3. Creez votre propre CV : cliquez en haut a gauche sur + Nouveau (voir section 4). Ou modifiez simplement la demo avec vos donnees.
  4. Remplissez les rubriques a gauche. L'apercu a droite se met a jour automatiquement.
  5. Choisissez un modele et une police qui vous plaisent (sections 7 et 8).
  6. Exportez le resultat en PDF (section 13).

3. L'interface en un coup d'oeil

L'application se compose de trois zones :

  • Barre superieure : c'est ici que vous gerez les CV (selection, Nouveau, Dupliquer, Renommer, Supprimer), choisissez le modele, la police et la couleur du texte, et trouvez les boutons d'import JSON, d'export JSON et d'export PDF. Tout a droite figure une indication d'enregistrement (par exemple "Enregistre").
  • Editeur (a gauche) : c'est ici que vous saisissez votre contenu. En haut se trouvent des onglets : Personnel, Competences, Experience, Formation, Projets, Langues, Divers.
  • Apercu (a droite) : affiche votre CV en temps reel sous forme de page A4. Des lignes en pointilles indiquent ou une page se termine et ou la suivante commence.
Enregistrement automatiqueVous n'avez rien a enregistrer manuellement. Des que "Enregistre" s'affiche en haut a droite, votre etat actuel est sauvegarde dans le navigateur.

4. Gerer plusieurs CV

Dans la barre superieure, tout a gauche, vous trouvez la zone CV avec un champ de selection et quatre boutons.

Creer un nouveau CV

  1. Cliquez sur + Nouveau.
  2. Saisissez un nom (par exemple "Candidature Data Engineer") et confirmez.
  3. Un CV vide est cree et devient immediatement actif. Remplissez-le maintenant.

Basculer d'un CV a l'autre

Ouvrez le champ de selection et choisissez le CV souhaite. L'application charge immediatement son contenu.

Dupliquer, Renommer, Supprimer

  • Dupliquer : cree une copie independante du CV actuel (par exemple pour essayer une variante).
  • Renommer : modifie le nom du CV actuel.
  • Supprimer : retire definitivement le CV actuel. Une demande de confirmation apparait. Le bouton est desactive lorsqu'il ne reste plus qu'un seul CV.
Attention a la suppressionLa suppression ne peut pas etre annulee. Si vous avez un doute, exportez le CV au prealable en JSON (section 14).

5. Modifier le contenu (toutes les rubriques)

Dans l'editeur a gauche, vous choisissez un onglet en haut. Chaque modification apparait immediatement dans l'apercu.

Personnel

Ici figurent vos donnees de base : nom, intitule de poste, adresse, e-mail, telephone, LinkedIn, GitHub, portfolio/site web, date de naissance, lieu de naissance, nationalite et situation familiale. Renseignez ce que vous souhaitez montrer. Les champs vides sont simplement omis dans l'apercu.

Competences (Skills)

  1. Cliquez sur + Ajouter une categorie (par exemple "Programmation").
  2. Cliquez sur Modifier au niveau de la categorie.
  3. Modifiez le nom de la categorie si necessaire.
  4. Saisissez une competence dans le champ en bas (par exemple "Python") et appuyez sur Entree ou sur le +. Repetez l'operation pour d'autres competences.
  5. Une competence peut etre supprimee via le petit x a cote de l'entree.
  6. Facultatif : reglez une note en etoiles par competence (selection a droite de la competence, 1 a 5 etoiles ou "sans"). Les etoiles sont affichees dans le modele "Profil Klassisch" ; les autres modeles les ignorent.

Experience (experience professionnelle)

  1. Cliquez sur + Ajouter une entree.
  2. Cliquez sur Modifier et renseignez le poste, l'entreprise, le lieu, De et A.
  3. Sous "Taches", ajoutez des puces individuelles avec + Point. Gardez-les courtes (voir section 16).

Formation

Comme pour l'experience : diplome, etablissement, lieu, De, A et remarques facultatives.

Projets

  1. Cliquez sur + Ajouter une entree, puis sur Modifier.
  2. Renseignez le titre du projet ainsi que le type/contexte.
  3. Facultatif : sous Lien de projet, saisissez l'adresse du projet (par exemple votre depot GitHub). Un bouton discret "Voir" apparait alors dans le CV, cliquable dans le PDF.
  4. Sous "Stack technique", ajoutez les technologies utilisees (saisie et Entree).
  5. Sous "Resultats", ajoutez de courtes puces.

Langues

Ajoutez la langue et le niveau (par exemple "Allemand" / "C1").

Divers (centres d'interet)

De courtes entrees comme les centres d'interet, le permis de conduire ou le benevolat.

Refermer les entreesEn cliquant a nouveau sur Modifier, vous repliez une carte. Cela garde une vue d'ensemble claire.

6. Importer une photo

  1. Allez dans l'onglet Personnel.
  2. Cliquez sur Choisir une photo et selectionnez un fichier image depuis votre appareil.
  3. Sous Affichage de la photo, choisissez entre Ronde et Carree.
  4. Avec le curseur Zoom de la photo, vous agrandissez le cadrage de l'image (100 a 250 pour cent), et avec Position verticale de la photo, vous deplacez le cadrage vers le haut ou vers le bas - ainsi le visage est parfaitement place dans le cadre. Le fichier d'origine reste inchange.
  5. Avec Supprimer, vous effacez de nouveau la photo.
Conseil pour la photoUtilisez une photo de portrait la plus carree possible et bien eclairee. L'image est enregistree directement dans le navigateur et n'est envoyee nulle part.

8. Choisir une police

Sous Police, vous choisissez parmi 11 polices (par exemple Inter, Arial, Calibri, Georgia, Times New Roman). L'apercu se met a jour immediatement. Les polices a empattements (Georgia, Times, EB Garamond) font classique, les polices sans empattement (Inter, Arial, Lato) font moderne.

9. Changer la couleur du texte

Par defaut, le texte est noir - c'est le plus lisible et c'est ce qu'attendent les recruteurs. Vous pouvez toutefois ajuster la couleur :

  1. En haut de la barre, cliquez sur le champ de couleur au niveau de Couleur du texte.
  2. Choisissez une couleur. L'apercu se met a jour immediatement.
  3. Avec le bouton Noir, vous remettez la couleur sur la valeur par defaut a tout moment.
Attention a la lisibiliteLes couleurs tres claires (par exemple gris clair ou jaune) sont difficiles a lire sur du papier blanc et font peu professionnel. Restez sur le noir ou un ton fonce. Les couleurs d'accent des titres ne changent pas pour autant.

10. Portfolio, LinkedIn, GitHub et liens de projet

  • Portfolio/Site web : si vous saisissez un site web dans l'onglet Personnel, il apparait sous forme d'un bouton "Portfolio" bien visible - il attire l'oeil au lieu de se perdre comme une longue adresse.
  • LinkedIn et GitHub : sont affiches sous forme d'icones compactes (et non comme une longue adresse). Cela economise de la place et donne un rendu epure.
  • Liens de projet : voir section 5 (Projets). Ils apparaissent sous forme d'un petit bouton "Voir".

Tous ces liens restent cliquables dans le PDF exporte.

11. Reordonner par glisser-deposer

  1. Dans l'editeur, passez la souris au-dessus d'une carte (par exemple une experience professionnelle).
  2. Cliquez et maintenez la carte (sur l'icone de poignee a gauche) et faites-la glisser vers le haut ou vers le bas.
  3. Relachez a la position souhaitee. L'ordre dans le CV s'adapte immediatement.
RecommandationPlacez les entrees les plus importantes et les plus recentes en haut - elles sont lues en premier.

12. Pages A4 et sauts de page

L'apercu affiche votre CV sous forme de vraies pages A4. A chaque limite de page, vous voyez une discrete ligne en pointilles portant la mention "Page 2", "Page 3" et ainsi de suite.

  • Pour qu'aucune entree ne soit coupee au milieu du texte, l'application pousse automatiquement vers la page suivante les lignes qui tomberaient exactement sur une limite de page - dans tous les modeles.
  • Le PDF exporte correspond exactement a l'apercu - y compris ces sauts de page propres.
CV trop long ?Raccourcissez les puces, retirez les entrees plus anciennes ou reduisez le nombre de projets. Une a deux pages, c'est l'ideal.

13. Exporter en PDF

  1. Cliquez en haut a droite sur Exporter PDF.
  2. Patientez un instant (le bouton affiche "Veuillez patienter..."). L'application genere un PDF A4 de plusieurs pages.
  3. Le PDF est telecharge automatiquement. Le nom de fichier contient le nom de votre CV.
  • Les liens (e-mail, LinkedIn, GitHub, portfolio, projets) restent cliquables dans le PDF.
  • Le PDF correspond a l'apercu. Verifiez-le tout de meme une fois avant l'envoi.
Connexion internetUne connexion internet est necessaire pour l'export PDF et pour le chargement correct des polices.

14. Sauvegarde : export et import JSON

Avec les fichiers JSON, vous sauvegardez vos CV sous forme de fichier - la methode la plus fiable contre la perte de donnees et pour passer a un autre appareil.

Exporter (sauvegarder)

  1. Selectionnez en haut le CV souhaite.
  2. Cliquez sur Exporter JSON. Un fichier .json est telecharge.
  3. Conservez ce fichier en lieu sur (par exemple dans un dossier cloud).

Importer (restaurer)

  1. Cliquez sur Importer JSON.
  2. Selectionnez votre fichier .json.
  3. Un nouveau CV est cree a partir de celui-ci (les CV existants restent inchanges).
RecommandationExportez une fois votre CV termine en JSON et conservez le fichier. Vous pouvez ainsi le restaurer a tout moment sur n'importe quel appareil.

15. Ou mes donnees sont-elles enregistrees ?

C'est important a comprendre pour ne perdre aucune donnee :

  • Vos CV sont enregistres dans le stockage local de votre navigateur (LocalStorage), pas sur un serveur.
  • Le stockage est lie a un navigateur et a une adresse. Sur un autre ordinateur, dans un autre navigateur ou sous une autre adresse web, vous ne voyez donc pas vos donnees automatiquement.
  • Si vous effacez les donnees du navigateur (par exemple l'historique avec les donnees de sites web), vos CV sont egalement supprimes.
Comment eviter la perte de donneesExportez les CV importants en fichier JSON (section 14) et conservez-les. C'est la seule methode sure pour une sauvegarde durable et pour passer d'un appareil a l'autre.

Au premier demarrage dans un navigateur vide, l'application affiche un CV d'exemple afin qu'elle ne paraisse jamais vide. Vos propres donnees ne sont jamais ecrasees par cela.

16. Conseils pour un CV convaincant

  • Court et percutant : une idee par puce, idealement une ligne. Evitez les longs blocs de texte.
  • Oriente resultats : indiquez des resultats et des chiffres (par exemple "temps de chargement reduit de 60 pour cent").
  • La lisibilite d'abord : texte noir sur fond blanc, police claire, espacement suffisant.
  • Pertinence : placez les entrees les plus importantes en haut (glisser-deposer, section 11).
  • Longueur : une a deux pages suffisent le plus souvent. Souvent, moins c'est plus.
  • Verifier : controlez l'orthographe et le PDF termine avant l'envoi.

17. Questions frequentes et resolution de problemes

Sur le site publie, mon CV est vide. Pourquoi ?
Vos donnees se trouvent dans le stockage local exactement du navigateur et de l'adresse ou vous les avez creees (voir section 15). Le site web possede son propre stockage, vide au depart. Solution : exportez votre CV en JSON la ou il est present, et importez-le sur le site (section 14).
Mes donnees ont disparu. Sont-elles perdues ?
Le plus souvent non. Verifiez que vous utilisez le meme navigateur et la meme adresse que lors de la creation. Verifiez le champ de selection en haut - peut-etre qu'un autre CV est simplement actif. Si vous disposez d'une sauvegarde JSON, importez-la. Si vous avez efface les donnees du navigateur, seules les donnees d'une sauvegarde JSON peuvent etre restaurees.
Le PDF coupe du contenu.
Les lignes individuelles qui tomberaient sur une limite de page sont automatiquement poussees vers la page suivante dans tous les modeles. Si malgre tout quelque chose se coupe de maniere genante, raccourcissez l'entree concernee ou deplacez-la par glisser-deposer.
Les icones LinkedIn ou GitHub ne sont pas visibles dans le PDF.
Les icones adaptent leur couleur a la couleur du texte et sont colorees correctement lors de l'export. Si une icone n'apparait pas sur un fond sombre, verifiez que l'adresse est correctement saisie, et exportez de nouveau.
Puis-je modifier le meme CV sur plusieurs appareils ?
Pas automatiquement, car les donnees sont stockees localement dans le navigateur. Exportez le CV en JSON et importez-le sur l'autre appareil (section 14).
Dois-je enregistrer ?
Non. L'application enregistre automatiquement. Lorsque "Enregistre" s'affiche en haut a droite, votre etat est sauvegarde. Pour une sauvegarde durable, utilisez en plus l'export JSON.
Le texte parait gris et difficile a lire.
En haut, cliquez sur Noir au niveau de la couleur du texte pour retablir la couleur par defaut bien lisible (section 9).
